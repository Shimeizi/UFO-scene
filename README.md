# 🛸 Alentejo UFO Scene

A project developed for the **Computer Graphics** course.

## 📖 Overview

This project combines the design and implementation of an interactive 3D scene inspired by the Alentejo landscape.

The project began with scene sketches, technical drawings, and scene graph design, followed by the implementation of a Three.js application featuring procedural terrain generation, textured environments, dynamic lighting, custom polygon meshes and an animated UFO.

## ✨ Features

* ✏️ Scene sketches and technical drawings
* 🏗️ Scene graph design
* 🌄 Procedural terrain generation
* 🌼 Procedural flower field texture
* ⭐ Procedural starry sky texture
* 🌳 Cork oak tree modeling
* 🏠 Low-poly Alentejo house
* 🛸 Animated UFO
* 💡 Directional, point and spotlight lighting
* 🎨 Multiple shading models
* 🌐 VR stereoscopic camera support

## 🛠️ Built With

* JavaScript
* Three.js

## 🎮 Controls

### Lighting

* `D` → Toggle directional light
* `P` → Toggle UFO point lights
* `S` → Toggle UFO spotlight

### Materials

* `Q` → Lambert shading
* `W` → Phong shading
* `E` → Toon shading
* `R` → Toggle lighting calculations

### Camera

* `7` → Perspective camera

### UFO Movement

* Arrow Keys
